
🚗 Vehicle Management System

A dynamic web-based Vehicle Management System developed using JSP, Servlets, and MySQL. The system allows users to browse and book vehicles (cars and bikes), while providing an admin panel to manage bookings and vehicle records efficiently.

🔹 Features
🚘 Browse available cars and bikes
📄 View vehicle details (model, price, availability)
🟢 Book Now option to reserve a vehicle
🔄 Real-time update of vehicle booking status
❌ Prevents duplicate bookings
👨‍💼 Admin Panel:
Accept or reject booking requests
Edit booking details
Add, update, and delete vehicles
Manage vehicle availability

🔹 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: JSP (Java Server Pages), Servlets
Database: MySQL
Server: Apache Tomcat

🔹 How It Works
User visits the system and views available vehicles
Selects a car or bike and clicks Book Now
Booking request is stored in the MySQL database
Admin reviews the request via the admin panel
Admin approves/rejects the booking
Vehicle availability updates automatically
